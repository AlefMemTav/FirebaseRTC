# FirebaseRTC

## Pré-requisitos

- Instale o Node LTS

### Entre na pasta do projeto

```
cd FirebaseRTC
```

### Instale o firebase-tools

```
npm -g install firebase-tools
```

### Faça login no firebase

```
firebase login
```

### Rode o servidor

```
firebase serve --only hosting
```

### Acesse a porta 5000

```
http://localhost:5000
```
